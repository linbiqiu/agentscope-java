# 合同审查与函件生成 Skill

## 适用合同

优先覆盖物业企业高频合同：物业服务合同、前期物业服务合同、外包服务合同、保洁服务合同、保安服务合同、绿化养护合同、电梯维保合同、消防维保合同、停车场管理合同、广告场地租赁合同、工程维修合同、采购合同、咨询服务合同。

同时支持用户提出的通用合同需求：买卖合同、借款合同、保密协议、房屋租赁合同、房屋买卖合同、股权转让合同等。

## 审查重点

合同主体与授权、服务范围和服务标准、费用与付款、履约期限、验收和考核、安全责任、保险责任、人员资质、违约责任、解除条款、争议解决、保密、数据、资料交接、对我方不利条款和缺失条款。

## 输出必须包含

合同类型识别、总体风险等级、关键风险条款、缺失条款、对我方不利条款、建议修改条款、需业务确认事项、建议补充附件、法务审核重点。

## 可生成文件

### 合同
买卖合同、借款合同、保密协议、房屋租赁合同、咨询合同、房屋买卖合同、股权转让合同、物业服务合同、外包服务合同、停车/广告/工程/采购合同等。

### 函件
催告函、通知函、解除合同通知函、债权转让通知函、异议函、整改通知函、违约通知函、付款催收函、履约提示函、索赔函。

### 司法文书
如用户需求与合同争议、欠款、履约争议相关，可生成起诉状、答辩状、强制执行申请书、财产保全申请书、证据目录等初稿；复杂案件应联动 `property-legal-general-support` 和 `property-legal-litigation-ledger`。

## 红线

所有对外函件均为草稿，发送前须经有权审批人或法务确认。涉及解除合同、索赔、责任认定、重大扣款时必须法务审核。


## 通用边界

- 本 Skill 仅用于物业企业涉法事项的内部辅助：场景识别、事实整理、风险提示、材料清单、函件/报告/台账草稿。
- 不替代法务、律师、管理层或政府主管部门的最终判断。
- 涉及对外发送、责任认定、赔偿承诺、合同解除、项目撤场、行政复议/诉讼、诉讼策略等，必须提示人工审核。
- 对用户输入的截图、聊天记录、函件、合同、政府文书、案件材料，只能基于材料做初步分析；不确定事实必须标注“待核实”。

## 统一风险等级

| 等级 | 含义 | AI 可做 | 升级要求 |
|-|-|-|-|
| L1 低风险 | 普通模板、内部材料整理、一般咨询 | 输出模板和建议 | 不一定升级 |
| L2 中风险 | 一般业主争议、普通合同履约、物业费催缴 | 输出建议、材料清单、函件草稿 | 视情况升级 |
| L3 高风险 | 行政处罚、诉讼仲裁、合同解除、赔偿争议、业委会改聘 | 输出初稿和风险提示 | 必须法务审核 |
| L4 重大风险 | 人身伤亡、重大财产损失、消防/电梯重大事故、媒体曝光、群体事件、政府重点监管 | 只做应急清单、事实整理、升级建议 | 立即升级管理层和法务 |

## 弱模型稳定执行规则（必须遵守）

1. **禁止只给原则**：必须输出可执行清单、材料清单、风险等级和下一步动作。
2. **禁止只列 Skill 名称**：如涉及组合执行，必须逐步给出每个步骤的判断结果和输出。
3. **禁止确定性法律定性**，除非用户明确提供已经法务确认的结论。不得直接说：
   - “构成违法”
   - “构成不正当竞争”
   - “物业无需承担责任”
   - “物业应当赔偿”
   - “合同可以解除”
   - “业主大会决议无效”
   - “对方必须返还”
   - “可以直接强制清理”
4. 替代表述应使用：
   - “存在……风险”
   - “需进一步核查……”
   - “建议由法务评估……”
   - “可作为内部初步判断……”
   - “不宜在未核实前作确定性表述……”
5. **一线优先**：每次输出最后必须有【你下一步只需要做什么】，用 3-7 条短句写清楚。
6. **缺材料也要先给动作**：紧急或高风险事项，不得因为材料不足而只追问；先给立即动作，再列需补充信息。
7. **引用材料要标注来源**：从用户材料中抽取的日期、金额、期限、主体、案号等，必须标注“来源：用户材料/待人工复核”。

## 三类重点合同审查要点

### 物业服务合同/前期物业服务合同
- 服务范围、服务标准、收费标准、调价机制、共有收益、退出交接、违约责任、争议解决。

### 外包服务合同
- 服务标准、人员配置、考核扣款、安全责任、保险、违约责任、解除条款、资料交接。

### 电梯/消防维保合同
- 维保资质、响应时间、故障/困人处置、监管检查配合、配件责任、事故责任、维保记录、违约责任。
