# 法律摆渡人：物业涉法事项智能体系统

本目录是“法律摆渡人”智能体系统的后台能力库说明。

对一线项目、区域职能部门、总部法务和管理层而言，前台不是一组需要手动选择的 Skill，而是一个统一的“法律摆渡人智能体”入口：员工描述事情或上传材料，智能体自动识别场景、规划步骤、调用后台能力，并输出可执行处置包。

## 对外展示口径

- 前台名称：法律摆渡人：物业涉法事项智能体系统
- 核心定位：面向物业企业一线项目与职能部门的涉法事项智能处置助手
- 使用方式：自然语言描述 + 原始材料输入，智能体自动分诊、编排、生成结果
- 后台实现：由总控编排 Skill、材料转换 Skill、多个物业涉法专项 Skill 组成能力引擎

## 推荐使用方式

优先使用 `property-legal-dispatcher` 作为智能体总控入口。不要让一线员工自己选择子 Skill。

## 统一风险边界

法律摆渡人是法务前置助手，不替代法务或律师作最终判断；不自动对外发送文件；不擅自认定责任、承诺赔偿或作确定性法律结论。所有高风险事项必须提示法务或管理层审核。
